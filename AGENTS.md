# AGENTS.md

This repository contains a research-oriented virtual ECU prototype in C for a PhD project in automotive systems.

Rules:
- Use C, not C++.
- Keep the project modular and lightweight.
- Do not add unnecessary dependencies.
- Keep code compatible with gcc on Linux/WSL.
- Preserve separation between sensors, control, actuators, diagnostics, safety monitor, fault injection, and logging.
- Prefer CSV logging for experiments.
- Keep the project suitable for academic research and paper writing.
